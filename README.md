# SoundSketch

Hardware and software for Soundsketch guitar pedal as shown on [Youtube](https://youtu.be/grorq3XL0zQ?si=FIOID7Fv4AuRCGD6)

Borrows from the open source [DaisySeedProjects](https://github.com/bkshepherd/DaisySeedProjects) pedal, but with midi and expression, and 3 way toggles, and screen/encoder removed. 

Not all of the software modules have been tested.
