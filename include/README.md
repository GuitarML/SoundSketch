# Include
This contains the SoundSketch.h header along with custom headers for SoundSketch software.

## Contents
 - expressionHandler: Expression pedal code for the SoundSketch board. Remaps knob controls to use with an expression pedal.

 - SoundSketch.h : Include header file for SoundSketch.  This provides an abstraction for the knobs, switches, and LEDs when writing software for SoundSketch.```
